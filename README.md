## Installation

1. Clone the repo
```sh
git clone https://github.com/PPetkov2000/Instagram-clone
```
2. Install NPM packages
```sh
npm install in the root folder
```
3. Run Application
```sh
npm start
```
4. Go to
```sh
http://localhost:3000/
```
